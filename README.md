# Getting started.

## install node deps
```sh
(cd react-app && npm i) && (cd electron-app && npm i)
```

## development
```sh
# from one shell
(cd react-app && npm start)
```

```sh
# from another shell
(cd electron-app && npm start)
```

## production
```sh
# from one shell
(cd react-app && npm run build) && (cd electron-app && npm run make)
```
