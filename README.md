# Getting started.

## install node deps
```sh
(cd react-app && npm i) && (cd electron-app && npm i)
```

## development

to run the server
```sh
# from another shell
(cd ../smack-be && npm run dev)
```

react app
```sh
# from one shell
(cd react-app && npm start)
```

electron env
```sh
# from another shell
(cd electron-app && npm start)
```



## production
```sh
# from one shell
(cd react-app && npm run build) && (cd electron-app && npm run make)
```
